# Contributing to Salaty

Thank you for considering contributing to Salaty! 🕌

## Getting Started

1. Fork the repository on GitHub
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/salaty.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

```bash
npm run dev   # Start dev server at http://localhost:3000
npm run build # Build for production
```

## Code Style

- Keep components clean and readable
- Use meaningful variable names
- Comment complex logic (prayer time math, notification scheduling)

## Submitting Changes

1. Ensure the app builds successfully: `npm run build`
2. Commit with a clear message: `git commit -m "feat: add weekly prayer statistics"`
3. Push and open a Pull Request against `main`

## Ideas for Contributions

- Weekly/monthly prayer statistics
- Qibla direction compass
- Adhan audio playback
- Multiple location support
- Offline-first improvements
- Accessibility (a11y) improvements
- RTL layout support for Arabic UI
